# Contributing to Budzie

This repo is public, but not open season.

## Gate

New issues and PRs are auto-closed by default.

Maintainers approve contributors from issue comments:

- `lgtmi`: future issues stay open
- `lgtm`: future issues and PRs stay open

Open an issue first. A PR without prior approval will be closed.

## Issue Bar

Keep issues short and concrete:

- State the bug or request.
- Explain why it matters.
- Include reproduction steps for bugs.
- Say whether you want to implement it.

## PR Bar

Every PR must:

- Link an approved issue.
- Explain what changed in plain language.
- Include tests or a reason tests do not apply.
- Keep scope narrow.

Do not submit code you cannot explain.
